# Nesneye-Dayal-Programlama-Vizesi

Bu proje Java tabanlı basit bir konsol uygulamasıdır. Verilerin kalıcı olması için SQLite veritabanı kullanılmıştır.

bu projede yapılabilen işlemler:

Kitap İşlemleri: Yeni kitap ekleme ve mevcut kitapları listeleme.
Öğrenci İşlemleri: Yeni öğrenci kaydı ve öğrenci listesi görüntüleme.
Ödünç alma işlemleri: Hangi öğrencinin hangi kitabı ne zaman aldığını kaydetme.
İade Sistemi: Ödünç alınan kitabın geri teslim tarihini güncelleme.
Veritabanı: Program kapatılsa bile veriler `SmartLibrary.db` dosyasında saklanır.


kullanılan sistemler:
Java 
SQLite
JDBC
